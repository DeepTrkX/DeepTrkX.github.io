Deep.TrkX

Deep.TrkX (Deep Learning for Tracking) is a project launched at Uppsala University which is mandated to explore deep use of deep learning for event and track reconstruciton in PANDA Experiment (See below). Some of the techniques being used are Dense Network (DNN), Covulutional (CNN) and Recurrent Networks (RNN).
